# Retail Sales Intelligence: Exploratory Data Analysis

## Project Overview

This project performs an end-to-end exploratory data analysis (EDA) on a retail sales dataset to identify revenue trends, top-performing products, and business insights.

The analysis was conducted using Python with Pandas, Matplotlib, and Seaborn.

---

## Dataset

The dataset contains retail transaction data including:

- Order details
- Product information
- Sales
- Profit
- Region
- Customer segment

Total Records: **9994**

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Git
- GitHub
- VS Code

---

## Key Analysis Performed

1. Total Revenue Calculation
2. Sales by Category
3. Monthly Sales Trend
4. Top 10 Best Selling Products
5. Profit by Category
6. Region Wise Profit
7. Loss Making Products

---

## Key Business Insights

- Technology category generates the highest sales and profit.
- Some furniture products generate losses due to high discount rates.
- West region contributes the highest overall profit.
- Sales show seasonal patterns across months.
